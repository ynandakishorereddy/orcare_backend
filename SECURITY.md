# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within ORCare, please send an e-mail to the project maintainers rather than creating a public issue. We will review the vulnerability and respond as quickly as possible.

We ask that you do not share the vulnerability publicly until a patch is released.
